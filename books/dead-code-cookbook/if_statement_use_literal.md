``` python:after_return.py
if true:
    pass
else:
    print("Am I dead?")

if false:
    print("Am I dead?")
```

# 🧪デッドコードレシピ🧪

if文の条件にリテラルを使用し、
条件分岐を行うifステートメントで、偽と判定されるリテラル表記をおこない、
その分岐中にコードを記載する。
